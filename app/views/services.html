Services

They give the controller additional functionality such as:

  • Fetching JSON data from a web service with $http
  • Log messages to the Javascript console with $log
  • Filter an array with $filter service

$http Service

The $http Service is used to make async requests to a server

You can use $http as a function with an options object:

$http({ method: 'GET', url: 'example.json'});

You can use one of the shortcut methods:

$http.get({'example.json' , {apiKey:'key'}});

These call return a Promise object with .success() and .error

You can configure $http to fetch JSON, the result will be automatically decoded into Javascript objects and arrays

data: {{data}} data status: {{data.status}}

In addition to get() requests, $http can post(), put() and delete()
